# 光线追踪实验

**注意**：图像加载需要时间

## 📋 项目简介

这是一个使用 Taichi 库实现的交互式光线追踪渲染系统，展示了经典的 Whitted-Style 光线追踪模型，包括硬阴影、理想镜面反射和玻璃材质折射效果。

## ✨ 功能特性

1. **Whitted-Style 光线追踪**：实现了基于迭代的光线弹射算法
2. **硬阴影效果**：通过阴影射线实现真实的硬阴影
3. **理想镜面反射**：完美反射效果，支持多次反射
4. **玻璃材质折射**：引入斯涅尔定律，实现透明玻璃球效果
5. **MSAA 抗锯齿**：像素内随机采样，平滑边缘过渡
6. **交互式参数调节**：实时调节光源位置和渲染参数

### 技术发现

1. **玻璃折射效果**：
   - 玻璃球中心部分呈现倒立像
   - 边缘部分由于入射角较大，反射成分更多
   - 全反射处理确保光线在特定角度下正确反射

2. **硬阴影实现**：
   - 阴影射线的发射和检测是实现硬阴影的关键
   - 阴影中的点只计算环境光，使阴影区域明显暗于其他区域
   - 光源位置对阴影效果影响显著

3. **MSAA 抗锯齿**：
   - 增加采样数可显著改善边缘质量
   - 建议采样数 4-8 以获得平衡效果
   
## 📄 技术文档

### 光线投射 vs 光线追踪

| 特性 | 光线投射 (Ray Casting) | 光线追踪 (Ray Tracing) |
|------|----------------------|----------------------|
| 射线类型 | 仅主射线 | 主射线 + 次级射线 |
| 光照计算 | 仅直接光照 | 直接光照 + 全局光照 |
| 反射/折射 | 不支持 | 支持 |
| 阴影 | 不支持 | 支持 |
| 复杂度 | 低 | 高 |

### Whitted-Style 光线追踪模型

本实验采用经典的 Whitted-Style 光线追踪模型。当一条主光线从摄像机出发击中物体表面时：

1. **阴影测试**：从交点向光源方向发射一条暗影射线
2. **材质分支**：根据材质类型决定光线后续行为

### 反射向量公式

$$\mathbf{R} = \mathbf{L}_{in} - 2(\mathbf{L}_{in} \cdot \mathbf{N})\mathbf{N}$$

其中：
- $\mathbf{L}_{in}$ 为入射光线方向
- $\mathbf{N}$ 为表面法向量

### 斯涅尔定律（折射）

$$\eta_1 \sin\theta_1 = \eta_2 \sin\theta_2$$

- 空气折射率 $\eta_{air} = 1.0$
- 玻璃折射率 $\eta_{glass} = 1.5$
- $\theta_1\$：入射角，入射光线与表面法线的夹角
- $\theta_2\$：折射角，折射光线与表面法线的夹角
### 阴影 Acne 修复

将射线起点沿法线方向偏移极小值：

$$\mathbf{P}_{new} = \mathbf{P} + \mathbf{N} \times \epsilon$$

其中 $\epsilon = 1e-4$
